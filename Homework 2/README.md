#Homework 2: 浮點數字序列最大可能值

Problem Description:
給一浮點數字的序列。插入 +，* 和括號以便獲得最大可能值。

Input:
第一個為數列個數，接下來為浮點數字的序列。

Output:
最大可能值。

Example:
假設該序列是 1.5, 2.0, 2.5, 1.0。選項包括：

(1.5 * 2.0) + (2.5 * 1.0) = 5.5<br>
1.5 + 2.0 + 2.5 + 1.0 = 7.0<br>
1.5 * 2.0 * 2.5 * 1.0 = 7.5<br>
(1.5 + 2.0) * (2.5 + 1.0) = 12.25
